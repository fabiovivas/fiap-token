## Instalação Local
Instruções para compilar e executar o frontend localmente

```bash
# clonar o repositorio
git clone  https://github.com/fabiovivas/fiap-token.git

# instalar
npm install

# subir o servidor
node server

# acessar pelo browser
http://localhost:3300/
```
